# INTERACTIVE RECOMMENDATIONS SYSTEM
## Tappable Practice Cards Instead of Text

---

## 🎯 THE PROBLEM

**Current (Text-based):**
```
AI: "Based on what you've shared, here are some gentle ways to reconnect:
• A 5-minute contemplative walk
• A short prayer while making coffee  
• Sitting in silence for 2 minutes

What feels most right for you today?"
```

**Issues:**
- User has to type their choice
- Friction in trying a practice
- No visual appeal
- Can't track what they chose

---

## ✨ THE SOLUTION

**New (Interactive Cards):**

AI says: "Here are some gentle ways to reconnect:"

Then shows 3 tappable practice cards:

```sql
-- Recommendations shown to user (for tracking)
CREATE TABLE recommendation_cards (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id) ON DELETE CASCADE,
  conversation_id UUID REFERENCES conversations(id),
  practice_type VARCHAR(100),
  title VARCHAR(200),
  description TEXT,
  duration VARCHAR(50),
  instructions TEXT,
  icon_emoji VARCHAR(10),
  shown_at TIMESTAMP DEFAULT NOW(),
  clicked BOOLEAN DEFAULT false,
  clicked_at TIMESTAMP,
  completed BOOLEAN DEFAULT false,
  helpful_rating INTEGER
);

CREATE INDEX idx_rec_cards_user ON recommendation_cards(user_id, shown_at DESC);
```

```javascript
/**
 * Generate recommendation cards (not text)
 */
async generateRecommendationCards(userId, currentMessage, userPersona, emotionalState) {
  try {
    const recommendations = await this.generateRecommendations(
      userId, 
      currentMessage, 
      userPersona, 
      emotionalState
    );

    // Transform recommendations into card format
    const cards = recommendations.recommendations.map(rec => ({
      practice_type: rec.practice,
      title: rec.practice,
      description: rec.why,
      duration: rec.duration,
      instructions: rec.instructions,
      icon_emoji: this.getIconForPractice(rec.practice),
      when: rec.when
    }));

    return cards;

  } catch (error) {
    console.error('Card generation error:', error);
    return [];
  }
}

/**
 * Map practice to appropriate emoji
 */
getIconForPractice(practice) {
  const lowerPractice = practice.toLowerCase();
  
  if (lowerPractice.includes('walk')) return '🚶‍♂️';
  if (lowerPractice.includes('prayer') || lowerPractice.includes('pray')) return '🙏';
  if (lowerPractice.includes('silence') || lowerPractice.includes('stillness')) return '🧘';
  if (lowerPractice.includes('journal')) return '📝';
  if (lowerPractice.includes('breath')) return '💨';
  if (lowerPractice.includes('scripture') || lowerPractice.includes('bible')) return '📖';
  if (lowerPractice.includes('coffee') || lowerPractice.includes('tea')) return '☕';
  if (lowerPractice.includes('nature') || lowerPractice.includes('outside')) return '🌳';
  if (lowerPractice.includes('music') || lowerPractice.includes('worship')) return '🎵';
  if (lowerPractice.includes('friend') || lowerPractice.includes('talk')) return '💬';
  
  return '✨'; // Default
}
```

```javascript
/**
 * Track card click
 */
exports.clickRecommendationCard = async (req, res) => {
  try {
    const { cardId } = req.params;
    const userId = req.user.id;

    // Verify card belongs to user
    const card = await db.query(
      `SELECT id FROM recommendation_cards 
       WHERE id = $1 AND user_id = $2`,
      [cardId, userId]
    );

    if (card.rows.length === 0) {
      return res.status(404).json({ success: false, error: 'Card not found' });
    }

    // Update click status
    await db.query(
      `UPDATE recommendation_cards 
       SET clicked = true, clicked_at = NOW() 
       WHERE id = $1`,
      [cardId]
    );

    // Also log to practices_log
    await db.query(
      `INSERT INTO practices_log (user_id, practice_type, suggested_by_ai)
       SELECT user_id, practice_type, true
       FROM recommendation_cards
       WHERE id = $1`,
      [cardId]
    );

    res.json({ success: true });

  } catch (error) {
    console.error('Click tracking error:', error);
    res.status(500).json({ success: false, error: 'Failed to track click' });
  }
};

/**
 * Mark recommendation as completed
 */
exports.completeRecommendation = async (req, res) => {
  try {
    const { cardId } = req.params;
    const { reflection } = req.body;
    const userId = req.user.id;

    // Update card
    await db.query(
      `UPDATE recommendation_cards 
       SET completed = true 
       WHERE id = $1 AND user_id = $2`,
      [cardId, userId]
    );

    // Update practices_log
    await db.query(
      `UPDATE practices_log 
       SET completed = true, reflection = $1
       WHERE user_id = $2 
       AND practice_type = (
         SELECT practice_type FROM recommendation_cards WHERE id = $3
       )
       AND created_at >= NOW() - INTERVAL '1 hour'`,
      [reflection, userId, cardId]
    );

    res.json({ success: true });

  } catch (error) {
    console.error('Complete tracking error:', error);
    res.status(500).json({ success: false, error: 'Failed to track completion' });
  }
};

/**
 * Rate recommendation
 */
exports.rateRecommendation = async (req, res) => {
  try {
    const { cardId } = req.params;
    const { rating } = req.body; // 1-5
    const userId = req.user.id;

    // Update card
    await db.query(
      `UPDATE recommendation_cards 
       SET helpful_rating = $1 
       WHERE id = $2 AND user_id = $3`,
      [rating, cardId, userId]
    );

    // Update practices_log
    await db.query(
      `UPDATE practices_log 
       SET helpful_rating = $1
       WHERE user_id = $2 
       AND practice_type = (
         SELECT practice_type FROM recommendation_cards WHERE id = $3
       )
       AND created_at >= NOW() - INTERVAL '1 hour'`,
      [rating, userId, cardId]
    );

    res.json({ success: true });

  } catch (error) {
    console.error('Rating tracking error:', error);
    res.status(500).json({ success: false, error: 'Failed to track rating' });
  }
};
```

## 🎯 HOW IT WORKS

### User Journey:

1. **User sends message** in recommendation phase
2. **AI responds** with short text: "Here are some gentle ways to reconnect:"
3. **Cards appear below** AI message (3 beautiful, tappable cards)
4. **User taps card** → Opens modal with full instructions
5. **User taps "Start Practice"** → Tracked as clicked & started
6. **User completes** → Can rate it (1-5 stars)
7. **Next time AI recommends** → Knows what worked before
