# COMPLETE AUTHENTICATION SCREENS

**YOU'RE RIGHT** - we need a proper signup that collects name/email before onboarding builds the persona.

## The Complete Flow

```
NEW USER:
WelcomeScreen → SignupScreen → Onboarding (8 questions) → TransitionPage → Chat

RETURNING USER:
WelcomeScreen → LoginScreen → Chat (skip onboarding)
```

---

## What to Build Next

Build these 4 screens in order:

### 1. WelcomeScreen
- App landing page
- "Get Started" button
- "Sign In" button
- Clean, inviting UI

### 2. SignupScreen ⭐ (MOST IMPORTANT)
- Collects: **Name**, Email, Password
- Name is used in AI responses
- Leads directly to onboarding
- Modern, easy-to-use form

### 3. LoginScreen
- Simple email/password
- "Forgot Password?" link
- Returns to main app

### 4. RootNavigator
- Routes users based on:
  - Not logged in → Auth screens
  - Logged in, no persona → Onboarding
  - Logged in with persona → Main app
