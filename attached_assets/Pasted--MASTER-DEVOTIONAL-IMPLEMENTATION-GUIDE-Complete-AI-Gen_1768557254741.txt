## 📊 MONITORING

### **Key Metrics to Track**

```sql
-- Daily active users completing devotionals
SELECT COUNT(DISTINCT user_id)
FROM user_devotional_progress
WHERE DATE(completed_at) = CURRENT_DATE;

-- Average streak length
SELECT AVG(current_streak)
FROM devotional_streaks;

-- Devotional completion rate
SELECT 
  COUNT(*) FILTER (WHERE is_completed = true)::float / COUNT(*) as completion_rate
FROM daily_devotional_assignments
WHERE assigned_date >= CURRENT_DATE - INTERVAL '7 days';

-- Most common themes in conversations
SELECT unnest(key_topics) as topic, COUNT(*)
FROM conversation_memories
WHERE created_at > NOW() - INTERVAL '30 days'
GROUP BY topic
ORDER BY COUNT(*) DESC
LIMIT 10;

-- AI generation costs
SELECT 
  COUNT(*) as devotionals_generated,
  COUNT(*) * 0.0007 as estimated_cost_usd
FROM devotionals
WHERE created_at >= DATE_TRUNC('month', CURRENT_DATE);
```

### **Devotional not generating**
```
Check: Is OPENAI_API_KEY set?
Check: Database migration ran?
Check: User has persona set?
Fix: Add console.logs in aiDevotionalGenerator.js
```

### **Memory extraction not working**
```
Check: Is conversation being marked as ended?
Check: memoryExtractor.extractMemories() being called?
Check: conversation_memories table exists?

```
